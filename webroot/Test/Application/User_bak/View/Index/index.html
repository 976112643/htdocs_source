旅行精选：

一路向西，漫行青海

我在星空下已诉说千遍，只愿在日出时与你再相见。

雄踞“世界屋脊”的青海省是个神秘而诱人的地方，青海的最佳旅游时间是短暂的夏季，即5～9月，冰消雪融，高原进入生机勃勃的时期。7月是青海的最热月，本人就是7月去的，虽是夏季但高原温差很大，特别是夜里看星空和早起看日出一定得做好保暖措施。

7月万亩油菜花在青海湖畔灿烂盛开，青海湖也呈现出最五彩缤纷的美态。此外位于湖西岸的黑马河正是观看星空和青海湖日出的最佳地点之一。

门源的油菜花在雪山下绵延几十公里，极富震撼力，铺天盖地的都是金黄色。

日月山东侧阡陌良田，一派塞上江南风光；西侧草原辽阔，牛羊成群，是一幅塞外景色。

茶卡盐湖是柴达木盆地有名的天然结晶盐湖，中国的“天空之镜”，可以赤脚走到湖面上观看和拍摄自己的倒影，如果是晴天的话夜里湖面还能映出星空的倒影。

无论摄影还是旅游，青海都是一个值得一去的地方。

最近和一个关系不错的朋友聊天，不知道为什么人只有站在状态外似乎才能懂得很多自己以前困惑很久的问题，每对分开的恋人都会在最后饱含真情地对对方说：“祝你幸福。”而事实上呢？对于不同的人而言幸福是不一样的，如果他爱你，愿意和你在一起，而最后未能如常所愿，那么即便她锦衣玉食也不会快乐。爱情也许不需要策马奔腾的轰轰烈烈，也不需要天上人间的你侬我侬，爱情和幸福一样是陪伴。就像那句很流行的话：“我所希望的幸福是每天睁开眼，你和阳光同在。”


人们常常爱对难以拥有的东西抱以幻想，好想下一秒就能遇见自己内心渴望的东西，后来又会因为得不到，而决定不再相信。曾经有人问我：“幸福是什么？”那个时候因为比较向往文艺，连说句话也会文绉绉的，感觉自己的人生就像一本书，结局怎样都不要紧，只要章节美丽，辞藻华丽，结局你让我怎么哭都行。所以那个时候一直觉得“幸福就是找个温暖的人过一辈子。”


年轻的时候我们喜欢偶像剧可能并不是因为男主多帅，而是可能因为曾经的某一个时刻我们也期待这样的爱情出现，每个女孩或许都希望自己能像公主一样被宠爱着，不为别的，只为我们自认为比起男生我们更专注爱情。后来见多身边人分分合合，也过了用耳朵听爱情的年纪，渐渐地觉得所谓的幸福可能不需要走在大街上系鞋带，也不需要像优乐美一样被捧在手心，也不需要穿越几座城市只为看一场烟花，幸福就是“一张桌子，三张板凳，四副碗筷。”
